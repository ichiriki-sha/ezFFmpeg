# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

---
## [1.0.0.0] - 2026-02-07

### Fixed

- 変換中にプレビューのメニューから開始位置、終了位置を設定できる不具合を修正

### Notes

- 本バージョンより ezFFmpeg は正式リリース版となります。

## [0.9.9.0] - 2026-01-25

### Fixed

- 出力タグの表示名に関する不具合を修正
- スプラッシュウィンドウの表示ロジックを改善

### Update

- 変換中にアプリケーションを終了した際のメッセージアイコンを変更
- 変換開始／終了項目が有効時の外枠表示を調整
- プロファイル一覧におけるプロファイル名の表示幅を調整

## [0.9.8.0] - 2026-01-17

### Added

- 起動時にスプラッシュウィンドウを表示するように追加

### Update

- アプリケーションアイコンを変更

## [0.9.7.0] - 2026-01-15

### Added

- 初期プロファイルを追加

### Fixed

- プレビュー再生中に動画をクリアした際、プレビュー表示が残る不具合を修正

### Update

- プレビューエリアと動画情報の間にスプリッターを追加し、高さを調整できるように改善
- プレビューエリアのスライダーバーをリニューアル
- プロファイルの初期化処理を見直し

## [0.9.6.0] - 2026-01-14

### Added

- プレビューエリアにコンテキストメニューを追加（再生／停止 など）

### Fixed

- プレビュー再生が最後まで到達した後、再度再生できない不具合を修正
- プレビュー停止中にも再生時間（ミリ秒）が変化してしまう不具合を修正
- ファイルダイアログでキャンセルした際に
  「アイテムを 0 件追加しました」と表示される不具合を修正

### Update

- 変換時のデフォルト出力先を「My Videos\ezFFmpeg」に変更
- プレビューエリアの再生時間をミリ秒単位で表示するように改善

## [0.9.5.0] - 2026-01-12

### Added

- 変換終了時間をリストビューに表示するように追加
- 変換終了時間を指定して変換できる機能を追加

### Fixed

- プレビューエリアで再生できない動画の場合に、エラーメッセージを表示するように改善
- その他、軽微な不具合を修正

### Update

- 処理完了時およびキャンセル時に表示していたメッセージボックスを廃止

## [0.9.4.0] - 2026-01-10

### Added

- 変換開始時間をリストビューに追加

### Update

- 変換開始時間を指定して変換できるように修正
- オプション画面の並列数のスクロールバーをボタンに変更

### Fixed

- 出力ファイル名生成時の不具合を修正

## [0.9.3.0] - 2025-12-29

### Update

- プリセット定義をタプルからクラスに修正
- 出力定義の拡張子をプログラムで設定するように修正

## [0.9.2.0] - 2025-12-28

### Update

- パラメータ画面の入力チェックを強化

## [0.9.1.0] - 2025-12-27

### Fixed

- パラメータの組み合わせでエラーになる不具合を修正

## [0.9.0.0] - 2025-12-26

### Added

- ezFFmpeg 初回公開
- GUI による動画・音声変換機能
- プロファイル管理機能
- GPU エンコード対応（NVENC / QSV / AMF）

### Notes

- 本バージョンは初期リリースのため、仕様が変更される可能性があります。
