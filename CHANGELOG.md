# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

---

## [0.9.5] - 2026-01-12

### Update

- プレビュー画面で再生できない動画の場合、エラーメッセージを表示するように修正
- 処理完了時／キャンセル時のメッセージボックスの表示を取りやめ

## [0.9.4] - 2026-01-10

### Update

- 出力ファイル名生成時の不具合を修正
- 変換開始時間をリストビューに追加
- 変換開始時間を指定して変換できるように修正
- オプション画面の並列数のスクロールバーをボタンに変更

## [0.9.3] - 2025-01-29

### Update

- プリセット定義をタプルからクラスに修正
- 出力定義の拡張子をプログラムで設定するように修正

## [0.9.2] - 2025-12-28

### Update

- パラメータ画面の入力チェックを強化

## [0.9.1] - 2025-12-27

### Update

- パラメータの組み合わせでエラーになる不具合を修正

## [0.9.0] - 2025-12-26

### Added

- ezFFmpeg 初回公開
- GUI による動画・音声変換機能
- プロファイル管理機能
- GPU エンコード対応（NVENC / QSV / AMF）

### Notes

- 本バージョンは初期リリースのため、仕様が変更される可能性があります。
