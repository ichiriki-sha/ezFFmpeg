# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

---

## [0.9.8.0] - 2026-01-17

### Update

- スプラッシュウィンドウを追加
- アイコンを変更

## [0.9.7.0] - 2026-01-15

### Update

- プレビューエリアと動画情報の間にスプリッターを設置し、プレビューエリアと動画情報の高さを変更できるように修正
- プレビューエリアで再生中に動画をクリアするとプレビューエリアが残る不具合を修正
- プレビューエリアのスライダーバーをリニューアル
- プロファイルの初期化方法を修正
- 初期プロファイルを追加

## [0.9.6.0] - 2026-01-14

### Update

- 変換時の出力先のデフォルトをMy Videos\ezFFmpegに変更
- プレビューエリアで最後まで表示後、再度再生できない不具合を修正
- プレビューエリアの時間をミリ秒まで表示するように修正
- プレビューエリアを停止しても時間(ミリ秒)が変わってしまう不具合を修正
- プレビューエリアにメニュー追加（再生／停止など）
- ファイルダイアログでキャンセル時、アイテムのを0件追加しましたとなる不具合を修正

## [0.9.5.0] - 2026-01-12

### Update

- プレビューエリアで再生できない動画の場合、エラーメッセージを表示するように修正
- 処理完了時／キャンセル時のメッセージボックスの表示を取りやめ
- 変換終了時間をリストビューに追加
- 変換終了時間を指定して変換できるように修正
- その他軽微な不具合を修正

## [0.9.4.0] - 2026-01-10

### Update

- 出力ファイル名生成時の不具合を修正
- 変換開始時間をリストビューに追加
- 変換開始時間を指定して変換できるように修正
- オプション画面の並列数のスクロールバーをボタンに変更

## [0.9.3.0] - 2025-12-29

### Update

- プリセット定義をタプルからクラスに修正
- 出力定義の拡張子をプログラムで設定するように修正

## [0.9.2.0] - 2025-12-28

### Update

- パラメータ画面の入力チェックを強化

## [0.9.1.0] - 2025-12-27

### Update

- パラメータの組み合わせでエラーになる不具合を修正

## [0.9.0.0] - 2025-12-26

### Added

- ezFFmpeg 初回公開
- GUI による動画・音声変換機能
- プロファイル管理機能
- GPU エンコード対応（NVENC / QSV / AMF）

### Notes

- 本バージョンは初期リリースのため、仕様が変更される可能性があります。
